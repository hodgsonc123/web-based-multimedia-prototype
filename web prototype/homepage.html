- the <HTML> tags define the root of an HTML document

- the <HEAD> tags define important information about the document

- the <BODY> tags define the documents body which is where the webpage's content will go

- there are so many tags but these are the foundations of every page!

- all tags must have a matching closing tag

CSS

What is CSS?

"CSS stands for Cascading Style Sheets with an emphasis placed on “Style.” While HTML is used to structure a web document (defining things like headlines and paragraphs, and allowing you to embed images, video, and other media), CSS comes through and specifies your document’s style—page layouts, colors, and fonts are all determined with CSS.

How Does It Relate To HTML?

Think of HTML as the foundation (every house has one), and CSS as the aesthetic choices (there’s a big difference between a Victorian mansion and a modern home)."- skillcrush.com
